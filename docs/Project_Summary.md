# Project Summary

## Objective

The objective of this project was to analyze the Procure-to-Pay (P2P) business process using Celonis EMS to identify bottlenecks, monitor KPIs, and improve process efficiency.

## Responsibilities

- Worked on Celonis EMS.
- Created PQL calculations for business KPIs.
- Built Procurement, Rework, Automation, and Benchmark dashboards.
- Performed process analysis and identified optimization opportunities.
- Used interactive dashboards for business insights.

## Outcome

The project provided visibility into procurement performance and helped analyze process inefficiencies using process mining techniques.
