# 📊 Procure-to-Pay (P2P) Process Mining using Celonis

## 📌 Project Overview
This project analyzes the Procure-to-Pay (P2P) business process using Celonis Execution Management System (EMS). The objective is to identify bottlenecks, improve process efficiency, monitor KPIs, and generate actionable business insights using Process Query Language (PQL).

---

## 🚀 Technologies Used

- Celonis EMS
- Process Query Language (PQL)
- Process Mining
- KPI Development
- Dashboard Design
- Data Modeling
- Data Integration

---

## 📈 Dashboard Features

- Procurement Overview
- Rework Analysis
- Automation Analysis
- Benchmark Analysis
- Throughput Time Analysis
- Vendor Performance
- Purchase Order Monitoring

---

## 📊 KPIs Implemented

- Net Order Value
- Purchase Orders
- Rework Rate
- Automation Rate
- Throughput Time
- Vendor Analysis
- Process Variants

---

## 🔍 Business Insights

- Identified process bottlenecks
- Monitored procurement performance
- Evaluated vendor efficiency
- Measured automation opportunities
- Reduced manual process analysis effort
